USB data
--------
To update:
	wget http://www.usb.org/developers/tools/comp_dump -O usb-vendors.txt

OUI data
--------
To update:
	curl http://standards.ieee.org/regauth/oui/oui.txt \
		| grep "^..-..-.." | sed "s/(hex)//" \
		| tr -s '\t' > oui.txt

Release Checklist
-----------------
	+ Update USB vendor list (see above)
	+ Check version numbers in Info.plist
	+ Run ./make_disk_image.sh
	+ Update appcast.xml
	+ SVN commit any changes
	+ Upload *.dmg and appcast.xml
